New Semester Setup

Use this order when you're setting up a fresh academic cycle so the system stays consistent.

1. Add departments if needed 2. Add subjects 3. Add faculty and HODs 4. Add students 5. Add fee records 6. Create exams and announcements

What Usually Causes Rework

Most cleanup later comes from skipping one of these checks while creating records.

  • Missing fee record after student creation
  • Wrong department while adding faculty or students
  • No HOD assigned to a department
  • Creating exams before subjects and semesters are ready

Controlled Student Onboarding

Public users do not get direct student accounts anymore. College admin shares a one-time link, the student submits through it once, and admin converts valid requests into real accounts.

Suggested Review Flow

Check the request, verify department and semester, then use the convert action to open a prefilled student form.

  • Pending requests are safer than open public registration.
  • College admin keeps control over the final student record.
  • Each invite link is one-time and tied to one email address.
  • This avoids wrong departments, bad data, and duplicate accounts.
