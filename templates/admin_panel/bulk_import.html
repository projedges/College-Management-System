How It Works
  1. Select whether you're importing students or faculty.
  2. Download the CSV template for your import type.
  3. Fill in the rows with your data (one record per line).
  4. Ensure dept_code matches an existing department code.
  5. Upload the file and confirm the import.
  6. The system skips rows where the username already exists.
